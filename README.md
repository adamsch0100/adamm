# AI Mass Marketing System

## What This Does
Generates AI tweets → Posts to multiple Twitter accounts → Drives traffic to digital products → Makes money

## Current Status
- ✅ AI tweet generation working (viral-tweet-generator.js)
- ✅ Database schema complete (Supabase)
- ✅ Frontend dashboard working
- ⏳ Multi-account posting (building now)
- ⏳ Auto-scheduling (building now)
- ⏳ Lead capture DMs (building now)

## How to Use (After Setup)
1. Open dashboard → Twitter page
2. Enter your niche/topic
3. Generate 50 tweets
4. Review → Keep best ones
5. Schedule across accounts
6. Monitor sales

## Tech Stack
- Backend: Node.js + Express (mcp-server.js)
- Frontend: Next.js (frontend/)
- Database: Supabase
- AI: OpenAI GPT-4 (viral-tweet-generator.js)
- Posting: Twitter API + MoreLogin

## Quick Start
1. Install: `npm install`
2. Setup Supabase (database config in .env)
3. Add OpenAI API key to Settings
4. Run: `npm run dev`
5. Go to http://localhost:3001

## Project Structure
- `mcp-server.js` - Backend API server
- `frontend/` - React dashboard
- `services/` - AI generation, posting, scheduling
- `database/` - SQL migrations
